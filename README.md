# Este é o meu portifólio

Muito obrigada por visitar esse repositório, meu portifólio foi construído em <strong>ReactJs</strong> juntamente com o <strong>framer motion</strong>.
Para a construção dele foram usados:

* React-icons;
* React Scroll;
* CSS Grid;
* Conceitos de responsividade;
* Context;
* Dark mode;
* Scroll to top;
* Framer motion;

## Veja o projeto completo:

https://portifolio-keitey.netlify.app/
